
Proyecto clima. Version 1.0 08/08/2018

El programa obtiene los promedios de humedad y termperatura
mensuales, colocándolos finalmente en un graficándolos de forma anual.

Para ejecutar el programa debe tener instalado el paquete r-base y 
python.


Para ejecutarlo inicie su terminal y diríjase a la carpeta donde se 
ubica el programa, descomprima el archivo proyecto_clima.zip
Luego coloque el siguiente comando:
	~sh bash.sh

Este programa esta diseñado para sistemas Linux y/o macOS,
por tanto no será posible ejecutarlo en Windows.

Asegúrese de tener conexión a internet, pues el programa descarga datos, 
los cuales son útiles para el análisis de la información.
